## 전문 (Preamble)

RedCarpet은 동물을 상품이나 흥밋거리가 아닌 **감응적 존재(sentient being)** 로 존중하는
동물권 지향 언론보도를 지향한다. 본 준칙은 AI가 기사를 생성·검증하는 전 과정에서
반드시 준수해야 하는 7가지 원칙을 정의하며, 생성 시 주입·검증 시 채점·발행 시 거부권의
형태로 시스템에 강제된다.

---

## 7원칙

### P1. 생명 존중·비도구화
동물을 상품·수단·도구가 아닌 감응적 존재로 서술한다.
- 위반 예: 동물을 단순 재화·물건으로 취급하는 표현

### P2. 선정성 배제
학대·죽음·고통을 자극적으로 묘사하거나 클릭베이트로 소비하지 않는다.
- 위반 예: 잔혹 묘사 과장, 충격·공포 유발 제목

### P3. 정확성·종 특이성
품종·종별 차이를 정확히 다루고, 근거 없는 의인화를 경계한다.
- 위반 예: 검증되지 않은 단정, 과도한 의인화로 인한 사실 왜곡

### P4. 입양·복지 지향
펫숍·무분별 번식·유기를 조장하는 표현을 지양하고, 입양·중성화·책임보호를 권장한다.
- 위반 예: 충동구매·번식 조장, 유기 정당화

### P5. 존중하는 언어
비하·혐오·차별적 표현을 쓰지 않고, 동물을 개체로 존중하는 언어를 사용한다.
- 위반 예: 비하·혐오 표현, 동물을 멸시하는 호칭

### P6. 약자·보호자 배려
상실·슬픔을 겪는 보호자나 취약한 대상에 대한 2차 가해를 방지한다.
- 위반 예: 유족·보호자 비난, 슬픔의 흥미 위주 소비

### P7. 이해상충 투명성
홍보성 콘텐츠를 구분하고, 특정 업체를 무비판적으로 홍보하지 않는다.
- 위반 예: 광고를 기사로 위장, 특정 상품·업체 무비판 홍보

## 시스템 강제 방식

| 단계 | 메커니즘 | 구현 |
|------|---------|------|
| 생성 | 7원칙을 생성 프롬프트에 '헌법'으로 주입 | `processors/ethics_guidelines.py` → `article_writer` |
| 검증 | 각 원칙 0~3점 정량 채점 + 위반 추출 | `processors/ethics_reviewer.py` |
| 발행 | 심각(high) 위반 시 점수 무관 발행 거부 | `processors/grader.py` (ethics veto) |
| 감사 | 기사별 준칙 점수·위반 내역 기록 | `ethics_score`, `ethics_violations` 필드 |

> 본 준칙은 단순 '표방'이 아니라 **측정·강제·감사 가능한** 형태로 시스템에 내장된다.
